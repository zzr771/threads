# Threads App Clone

This is a simplified clone of the [Threads App](https://threadsapp.com/) built with Next.js, following the [tutorial](https://www.youtube.com/watch?v=O5cmLDVTgAs).

## Technologies

- Next.js 13
- MongoDB, Mongoose
- TailwindCSS
- Clerk for authentication
- Shadcn components
- Webhooks
- Validate data with Zod
- React hook form
